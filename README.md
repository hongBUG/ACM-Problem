> 第3题，求最长递增子序列，这题比较坑的地方在于用普通的pd来写会超时，这题有O(nlogn)时间复杂度的解法：
>> 设一个辅助数组，遍历原数组，
>> 然后找到每个数组应该被插入的位置
>> 其中找到应该插入位置的过程使用O(logn)的算法来实现(如二分查找)
>> 故最终的时间复杂度为O(nlogn)
